# 💰 Gastos CLI

## 📌 Descrição
O Gastos CLI é uma aplicação simples desenvolvida em Python para ajudar pessoas a controlarem seus gastos do dia a dia diretamente pelo terminal.

## 🎯 Problema
Muitas pessoas têm dificuldade em controlar seus gastos e acabam perdendo o controle financeiro.

## 💡 Solução
Esta aplicação permite registrar, listar, remover e calcular gastos de forma simples e rápida via linha de comando.

## 👥 Público-alvo
- Pessoas que querem organizar suas finanças
- Estudantes
- Usuários que preferem soluções simples no terminal

## ⚙️ Funcionalidades
- Adicionar gasto
- Listar gastos
- Ver total de gastos
- Remover gasto

## 🛠️ Tecnologias utilizadas
- Python
- Pytest (testes)
- Ruff (lint)
- Git e GitHub
- GitHub Actions (CI)
